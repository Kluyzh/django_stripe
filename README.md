# Django Stripe Store

## Описание проекта
Сухой проект интернет-магазина с интеграцией Stripe для обработки платежей.
Поддерживает отдельные товары, заказы с несколькими товарами

## Особенности
- Интеграция Stripe Checkout для безопасных платежей

- Управление товарами через Django Admin

- Система заказов с поддержкой нескольких товаров

- Docker контейнеризация для простого развертывания

- Использование env для безопасной конфигурации

## Как запустить проект:
Клонировать репозиторий и перейти в него в командной строке:

```
git clone git@github.com:Kluyzh/django_stripe.git
```
Создать файл .evn для хранения ключей в корне проекта по подобию .env.example

Из основной директории
```
docker compose up -d
```
Создать суперпользователя
```
docker-compose exec web python manage.py createsuperuser
```
Зайти в админ зону
```
http://127.0.0.1:8500/admin/
```
В админ зоне создать пару товаров, создать заказ с несколькими товарами
Протестировать создание платежной сессии по путям
```
http://127.0.0.1:8500/items/1/
http://127.0.0.1:8500/items/2/
http://127.0.0.1:8500/order/1/
```
